# PROMPTS_USED.md

This file records the prompts used while building the Workflow Builder Lite project.
Responses from AI tools are intentionally not included.

AI tools used:
- ChatGPT (GPT-5.2)

The purpose of AI usage was to speed up scaffolding, validate approaches, and brainstorm edge-cases.
All generated code was manually reviewed, modified, and tested before committing.
